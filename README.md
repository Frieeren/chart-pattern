# chart-pattern
chart pattern similarity service

## 프로젝트 소개
현재 차트와 유사한 패턴을 가진 차트들을 검색하고 분석해주는 서비스입니다.
과거 데이터에서 차트 패턴을 찾아 분석에 도움을 제공합니다.

## 실행 가이드
### 1. project clone

```bash
$ git clone https://github.com/Frieeren/chart-pattern
$ cd chart-pattern
```

### 2. 실행

```bash
$ docker compose up -d
```

### 3. development server
http://localhost:80
